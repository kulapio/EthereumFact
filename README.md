# Pricing
- **20,000** gas for **1 Word** (8 bytes)
- **1 Word** cost **0.00008 Ether** (0.000000004 ether/gas).
- **1 kB** cost **.01 Ether**
- **1 MB** cost **10 Ether**
- **1 MB** cost **$8,600.00** ($860/ether)
